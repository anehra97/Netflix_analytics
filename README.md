# Netflix Streaming Analysis — SQL Project

## Overview

This project analyzes user engagement and content performance using a simulated **Netflix Streaming Dataset**.

The objective is to apply SQL analytics techniques to understand viewing behavior, subscription performance, and content engagement trends across countries, devices, and genres.

The project demonstrates how structured querying can transform raw streaming data into actionable business insights similar to real-world analytics used in media, entertainment, and subscription-based platforms.

---

## Business Objectives

The analysis answers key streaming analytics questions:

* Which countries generate the highest viewing hours?
* How do Premium, Standard, and Basic subscriptions perform?
* Which genres and content types drive engagement?
* How viewing behavior differs across devices and regions
* Identification of high-performing content segments
* Detection of low device usage patterns

---

## Dataset Description

The dataset `netflix_streaming_data` represents streaming activity and engagement metrics.

### Key Fields

* `user_name` — Viewer identifier
* `country` — Viewer location
* `language` — Content language
* `subscription_type` — Basic, Standard, Premium
* `content_title` — Title watched
* `content_type` — Movie or Series
* `genre` — Content genre
* `device_type` — Mobile, TV, Tablet, etc.
* `watch_date` — Viewing date
* `watch_hours` — Total watch time
* `rating` — Viewer rating
